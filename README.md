# Paw Paw: AI-Powered Pet Disease Diagnosis 🐾🩺

An AI model designed to provide preliminary diagnosis for pet health issues with high accuracy.

## 📊 Performance
- **Training Accuracy:** 99.4%
- **Validation Accuracy:** 97.2%
- **Model Result:** Successfully completed 30 epochs with optimal convergence.
